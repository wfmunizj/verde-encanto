# Verde Encanto

## Descrição
O projeto **Verde Encanto** é uma loja online dedicada à venda de produtos naturais, suplementos, nozes e sementes. O objetivo é oferecer uma experiência de compra simples e acessível, promovendo uma alimentação saudável.

## Estrutura do Projeto
- **index.html**: Página inicial da loja.
- **supplements.html**: Página dedicada a suplementos.
- **nuts.html**: Seção de nozes.
- **seeds.html**: Página para sementes.
- **css/**: Diretório contendo estilos CSS.
- **js/**: Diretório para scripts JavaScript.
- **img/**: Imagens utilizadas no site.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Font Awesome (para ícones)
- Google Fonts (Open Sans)
